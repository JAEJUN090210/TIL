## TIL - 06.28/Web - Electron

### Summary

- Chrominm/Node.js 기반의 데스크톱앱 제작을 주 기능으로 함

### Body

- OpenJS Foundation에서 관리하는 오픈소스 GUI 프레임워크

- Atom shell을 원본으로 두는 프레임워크

- 웹 앱과 데스크톱 앱을 거의 동일하게 제작할 수 있음

- 단일 코드베이스로 크로스플랫폼을 거의 완벽하게 지원함

-  내부적으로 Chromium 렌더링 엔진 + Node.js의 런타임을 사용

  

### (+)

- 웹 개발자의 앱개발 장벽을 낮춤

- 프론트엔드 코드를 거의 대부분 살려서 데스크톱앱 구현이 가능

- OS별 네이티브 API 접근 가능

  

### (-)

- Chromium 엔진이 거의 통으로 앱에 포함되므로 런타임이 무겁고 실행속도가 느림
  - WASM 활용을 통하여 성능 문제를 일부 해결할 수 있음
- 백엔드 구동에 이용되는 Node.js의 런타임에서 기본적으로 많은 메모리를 사용함
